# Agent Skills Repository

This repository collects reusable skill definitions for GitHub Copilot and other agent-based workflows.

## Repository contents

- `okf/`: the OKF skill package and its reference materials
- `llama-cpp-tuning/`: a skill for tuning llama.cpp performance on Apple Silicon

## How to use this repository

Import or reference the skill directories from this repository when you want to reuse their `SKILL.md` definitions in your own agent workflows.

## Notes

Some skills may include local reference files or helper scripts that are specific to that skill's domain.
